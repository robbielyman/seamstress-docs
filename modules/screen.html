Functions

clear ()
clears the screen.
set (value)
sets the screen which will be affected by following screen calls. call screen.reset to return to the previous state.

Parameters:

  • value integer 1 (gui) or 2 (params)
reset ()
resets which screen will be affected by future screen calls.
refresh ()
redraws the screen; reveals changes.
color (r, g, b, a)
sets screen color.

Parameters:

  • r integer red value (0-255)
  • g integer green value (0-255)
  • b integer blue value (0-255)
  • a integer alpha value (0-255) (default 255)
pixel (x, y)
draws a single pixel.

Parameters:

  • x integer x-coordinate (1-based)
  • y integer y-coordinate (1-based)
line (ax, ay, bx, by)
draws a line.

Parameters:

  • ax integer source x-coordinate (1-based)
  • ay integer source y-coordinate (1-based)
  • bx integer target x-coordinate (1-based)
  • by integer target y-coordinate (1-based)
rect (x, y, w, h)
draws a rectangle.

Parameters:

  • x integer upper-left x-coordinate (1-based)
  • y integer upper-left y-coordinate (1-based)
  • w integer width in pixels
  • h integer height in pixels
rect_fill (x, y, w, h)
draws a filled-in rectangle.

Parameters:

  • x integer upper-left x-coordinate (1-based)
  • y integer upper-left y-coordinate (1-based)
  • w integer width in pixels
  • h integer height in pixels
text (x, y, text)
draws text to the screen.

Parameters:

  • x integer upper-left x-coordinate (1-based)
  • y integer upper-left y-coordinate (1-based)
  • text string text to draw
get_text_size (text)
gets size of text.

Parameters:

  • text string text to size

Returns:

  1. integer w width in pixels
  2. integer h height in pixels
Screen. ()
returns the size of the current window.

Returns:

  1. integer w width in pixels
  2. integer h height in pixels
key (char, modifiers, is_repeat, state, window)
callback executed when the user types a key into the gui window.

Parameters:

  • char string or table either the character or a table of the form {name = "name"}
  • modifiers table a table with the names of modifier keys pressed down
  • is_repeat bool true if the key is a repeat event
  • state integer 1 for a press, 0 for release
  • window integer 1 for the main window, 2 for the params window
mouse (x, y, window)
callback executed when the user moves the mouse with the gui window focused.

Parameters:

  • x integer x-coordinate
  • y integer y-coordinate
  • window integer 1 for the main window, 2 for the params window
click (x, y, state, button, window)
callback executed when the user clicks the mouse on the gui window.

Parameters:

  • x integer x-coordinate
  • y integer y-coordinate
  • state integer 1 for a press, 0 for release
  • button integer bitmask for which button was pressed
  • window integer 1 for the main window, 2 for the params window
resized (x, y, window)
callback executed when the user resizes a window

Parameters:

  • x integer new x size
  • y integer new y size
  • window integer 1 for the main window, 2 for the params window
